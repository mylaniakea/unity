2. GET API TOKEN
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

TOKEN=$(curl -s -X POST http://localhost:8000/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"admin123"}' | jq -r '.access_token')

echo $TOKEN

Can't connect to API?
  - Check port forward: kubectl -n homelab port-forward svc/backend-service 8000:8000
  - Check pod status: kubectl -n homelab get pods | grep backend
  - View logs: kubectl -n homelab logs -f -l app=unity-backend

Auth failing?
  - Verify credentials: admin / admin123
  - Check token: echo $TOKEN (should be long JWT string)

API returning errors?
  - Check logs: kubectl -n homelab logs -f deployment/unity-backend
  - Verify database: kubectl -n homelab get pod postgres-0
